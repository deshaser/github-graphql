# Код ревью проекта

- Добавить виртуализацию списка репозиториев, например с помощью библиотеки @tanstack/react-virtual или react-virtual
- Отсутствует стратегия инвалидации кэша и нет оптимистических обновлений
- При масштабировании может потребоваться централизованное управление состоянием, например Zustand или Redux, а так же ленивая подгрузка
- Отсутствует документация компонентов, рекомендуется внедрить Storybook
